ADB Setup Instructions

Step 1: Factory Reset

Settings → System → Reset → Factory Data Reset

⚠️ Do NOT add Google account during setup

Step 2: Enable USB Debugging

Settings → About → Tap "Build Number" 7 times

Settings → Developer Options → USB Debugging ON

Step 3: Download APK

Download eden.apk

Step 4: Run ADB Commands

# Install APK
adb install eden.apk
# Set Device Owner
adb shell dpm set-device-owner com.eden.mkopa/.DeviceAdminReceiver
# Launch App
adb shell am start -n com.eden.mkopa/.MainActivity

✅ Done! Device is now locked and ready for customer login
